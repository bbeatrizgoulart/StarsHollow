## Stars Hollow Management System ☕️

Este é um projeto de estudo desenvolvido para praticar os conhecimentos adquiridos na disciplina de Linguagem de Programação II (UFSC), focado em aplicar conceitos de **Programação Orientada a Objetos (POO)** e organização de software em **C++**.

O sistema simula a gestão da cidade de Stars Hollow (da série *Gilmore Girls*), gerenciando diferentes estabelecimentos através de herança e polimorfismo. O objetivo é que o usuário tenha a experiência completa: se hospedar no hotel da Lorelai, fazer compras no Doose’s, tomar café no Luke e visitar o antiquário dos Kim.

## 🚀 Status do Projeto: Em Desenvolvimento

### Estabelecimentos Planejados: 
- [x] **Independence Inn**: Gestão de hóspedes e check-in. 
- [ ] **Luke's Diner**: Sistema de pedidos e fila. 
- [ ] **Doose's Market**: Inventário e estoque. 
- [ ] **Kim's Antiques**: Catálogo de itens únicos. 
- [ ] **Miss Patty’s**: Agenda de aulas e alunos. 
- [ ] **Stars Hollow Books**: Busca e indexação de livros.

## 🛠️ Tecnologias Utilizadas:
 - Linguagem C++ 
- Programação Orientada a Objetos (Herança e Encapsulamento)
